## Run

(Be sure to [install `bun`](https://bun.sh))

```
$ git clone https://github.com/bwasti/img2emoji.git
$ cd img2emoji
$ bun add @shumai/shumai
$ bun serve.ts
```

Then open `localhost:3000`

## Train from scratch

```
$ bun add @shumai/image
$ bun train.ts
```
